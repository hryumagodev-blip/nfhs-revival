# NFHS-India-Trends

Reviving and modernizing an abandoned 2017 data analysis project on India's National Family Health Survey (NFHS) data using Python, Pandas, NumPy, and statistical analysis.

Built using official government health survey data across 700+ Indian districts.

## Project Goal

This project aims to explore public health trends in India between 2015–2021 using NFHS data.

Instead of building a toy dataset project, the goal is to work with large-scale, real-world government data and understand how actual data analysis pipelines work:

* data cleaning
* preprocessing
* exploratory analysis
* statistical testing
* visualization
* feature engineering
* insight generation

Original abandoned repository:
https://github.com/Data4Democracy/india-nfhs

---

## Key Questions

This project investigates questions such as:

* Which Indian states improved the most in maternal healthcare?
* Did child vaccination rates improve nationally?
* Which districts declined despite national progress?
* Is there a measurable relationship between women's education and child nutrition?
* Which districts are the most vulnerable based on combined health indicators?
* How does my home state compare against the national average?
